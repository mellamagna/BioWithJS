For my previous employer, I'd written an extensive script from scratch designed to streamline the data entry process for employees in said department, designed to conserve keystrokes by reusing data keyed into previous redundant fields in the current form.

  • Written in a proprietary scripting language bundled into a Telnet terminal emulator (PowerTerm by Ericom Software). Limited documentation of said language (PowerTerm PSL) exists online, leading to a challenging but nonetheless successful debugging experience
  • Bound to a single hotkey for the end user's intuitive use, dynamically detecting which input page it was on via screen capturing methods and changing its functionality based on window header text and terminal cursor position
  • Deployed to 20+ data entry operators, improving productivity quota for the entire department
